#### ⚡ Power Commands (7 команд)

- ✅ **Screenshot** - JPEG скриншот (max 5MB)

#### ⌨️ Input Commands (7 команд)
- ✅ **Clipboard Read/Write** - Управление буфером обмена
- ✅ **Volume Mute/Unmute** - Отключение/включение звука

- ✅ **Volume Set** - Установить громкость на X%
- ✅ **CMD Execute** - Выполнить CMD команду (timeout 30s)

#### 📁 Files Commands (2 команды)
- ✅ **Launcher** - Запустить приложение
- ✅ **Search Files** - Поиск файлов по маске (max 100)
