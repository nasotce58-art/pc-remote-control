# 🖥️ Интеграция командной строки в приложение

## ✅ Описание добавленной функции

В раздел **⌨️ Управление** (Control) добавлена полноценная система ввода и выполнения команд:

### 📋 Компоненты:

1. **Текстбокс для команд** — поле ввода для ввода PowerShell/CMD команд
2. **Кнопка "Выполнить"** — запускает введённую команду
3. **История команд** — ListBox, показывающий всі выполненные команды и результаты

---

## 🔧 Технические детали

### Файлы, которые были изменены:
- `Views/DashboardWindow.xaml.cs` — добавлены методы обработки команд

### Добавленные методы:

#### `ExecuteCommand(string command)`
- Проверяет наличие команды
- Логирует команду в историю с временной меткой
- Вызывает PowerShell для выполнения
- Логирует результат и ошибки

#### `ExecuteCommandInPowerShell(string command)`
- Запускает процесс PowerShell
- Перенаправляет стандартный вывод и ошибки
- Логирует результаты выполнения в историю команд

### Новое поле класса:
```csharp
private ListBox? _commandHistoryBox;  // Ссылка на ListBox с историей
```

---

## 📝 Как использовать:

1. **Откройте приложение** → перейдите в раздел **⌨️ Управление**
2. **Введите команду** в текстбокс (например: `ipconfig`, `Get-Process`, `dir`)
3. **Нажмите кнопку "Выполнить"** (▶️ Выполнить)
4. **История команд** отобразит:
   - Время выполнения команды `[HH:mm:ss]`
   - Саму команду: `$ command`
   - Результат/ошибку: `> output` или `❌ error`

## 🔜 Следующие шаги для интеграции с Telegram:

### 1. **Создать обработчик для сообщений из Telegram**
   - Перехватывать сообщения от пользователей в `TelegramAPIClient.cs`
   - Проверять, если сообщение содержит символ команды (например, начинается с `@`)

### 2. **Маршрутизировать команды**
   - Если команда — простая PowerShell: выполнить через `ExecuteCommand()`
   - Если специальная команда (напр., `/status`): обработать отдельно

### 3. **Отправлять результаты в Telegram**
   - Перехватить результат из `_commandHistoryBox`
   - Отправить ответ пользователю в Telegram через `SendMessageAsync()`

### Пример интеграции:
```csharp
// В TelegramAPIClient.cs, после обработки сообщения:
if (messageText.StartsWith("@"))
{
    string command = messageText.Substring(1);  // Убираем @
    dashboardWindow.ExecuteCommand(command);    // Выполняем на ПК
    
    // Получаем последнюю запись из истории
    var result = _commandHistoryBox.Items[0]?.ToString();
    await SendMessageAsync(userId, $"✅ Результат:\n{result}");
}
```
