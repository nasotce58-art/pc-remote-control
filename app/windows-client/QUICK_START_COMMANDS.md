## 🚀 3 ШАГА ДЛЯ ЗАПУСКА:

### Шаг 1: Откройте приложение
```powershell
cd 'c:\conrol pc\app\windows-client'
dotnet run
```

Нажмите на пункт меню **⌨️ Управление и Ввод**

## 🤖 ИНТЕГРАЦИЯ С TELEGRAM:

### Когда будет подключена, можно писать:
```
Пользователь в Telegram: @ipconfig
Бот ответит: ✅ Результат выполнения команды:
            Windows IP Configuration ...
```

### Форматы команд из Telegram:
```
@command              — выполнить команду
/exec command         — выполнить команду (альтернатива)
```

```powershell
Введите: echo "Hello World"
Результат должен быть: Hello World
```

### Тест 2: Системная команда
```powershell
Введите: whoami
Результат должен быть: DOMAIN\username
```

```powershell
Введите: shutdown -s

```

## 🐛 РЕШЕНИЕ ПРОБЛЕМ:

### Проблема: Команда не выполняется
**Решение:** Убедитесь, что это действительная PowerShell команда

### Проблема: История команд пуста
**Решение:** Проверьте, что нажали кнопку "Выполнить"

### Проблема: Ошибка в истории
**Решение:** Проверьте синтаксис команды в PowerShell
