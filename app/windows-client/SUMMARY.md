## 🎯 Все реализованные команды (22 + все вариации)

### ⚡ Power Commands (7)

- [x] **restart** - Перезагрузка через 10 сек (отменяемо)
- [x] **shutdown** - Выключение через 10 сек (отменяемо)
- [x] **sleep** - Режим сна (standby)
- [x] **force_shutdown** - Принудительное выключение (отменяемо)
- [x] **monitor_on** - Включить монитор
- [x] **monitor_off** - Выключить монитор
- [x] **monitor_toggle** - Переключить состояние
- [x] **lock** - Заблокировать Windows
- [x] **timer** - Таймер выключения (1-1440 минут)

### 📊 Monitoring Commands (4 + вариации)

- [x] **system_stats** - CPU, RAM, Disk, Network, Internet
- [x] **process_list** - Все процессы с деталями
- [x] **kill_process** - Завершить процесс по имени или PID
- [x] **screenshot** - JPEG скриншот (quality 10-100, max 5MB)

### ⌨️ Input Commands (7)

- [x] **clipboard_read** - Прочитать буфер обмена
- [x] **clipboard_write** - Записать в буфер обмена
- [x] **volume_mute** - Отключить звук
- [x] **volume_unmute** - Включить звук
- [x] **volume_up** - Увеличить на 10%
- [x] **volume_down** - Уменьшить на 10%
- [x] **volume_set** - Установить на X% (0-100)
- [x] **cmd_execute** - Выполнить CMD команду (timeout 30s)

### 📁 File Commands (2)

- [x] **launcher** - Запустить приложение/файл
- [x] **search_files** - Поиск по маске (max 100)

### ⚙️ Settings Commands (2)

- [x] **autorun_enable** - Добавить в автозагрузку
- [x] **autorun_disable** - Удалить из автозагрузки
- [x] **about** - О программе и системе
