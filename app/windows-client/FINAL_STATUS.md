- ⏳ Telegram интеграция не подключена (только код готов)

### Решения:
```csharp
// Вариант 1: Сделать публичным
public void ExecuteCommand(string command) { ... }

// Вариант 2: Добавить public wrapper
public void ExecuteCommandPublic(string cmd) => ExecuteCommand(cmd);

// Вариант 3: Добавить в TelegramAPIClient
if (cmd.StartsWith("@"))
{
    await dashboardWindow.ExecuteCommand(cmd);
}
```
