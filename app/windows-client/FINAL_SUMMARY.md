## 🚀 КАК ИСПОЛЬЗОВАТЬ:

### Локально (в приложении):
```
1. Запустите приложение
2. Перейдите в ⌨️ Управление
3. Введите команду (например: ipconfig)
4. Нажмите Выполнить
5. Результат появится в истории команд

```

## 🔍 БЫСТРАЯ ПРОВЕРКА КОДА:

### Метод 1: ExecuteCommand
```csharp
// Валидирует, логирует и выполняет команду
// Результат появляется в _commandHistoryBox
ExecuteCommand("ipconfig");
```

### Метод 2: ExecuteCommandInPowerShell
```csharp
// Запускает процесс PowerShell
// Логирует результаты и ошибки
ExecuteCommandInPowerShell("Get-Process");
```

### Класс: TelegramCommandExecutor
```csharp
// Обрабатывает команды из Telegram
var executor = new TelegramCommandExecutor(dashboardWindow);
var result = await executor.ProcessTelegramCommand("@ipconfig", userId, username);
```
